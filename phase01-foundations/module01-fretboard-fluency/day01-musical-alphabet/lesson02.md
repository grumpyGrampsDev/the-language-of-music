---

## Objective

Understand that the musical alphabet is organized by distance.

The space between two notes is called an **interval**.

The smallest interval used in Western music is the **half step**.

On guitar:

> One fret = One half step.

---

## Lesson

The 12 notes are not a random list.

They form a repeating pattern called the **chromatic scale**.

```
A
A#/Bb
B
C
C#/Db
D
D#/Eb
E
F
F#/Gb
G
G#/Ab
```

After G#/Ab, the pattern begins again:

```
A
```

The distance between every note is one half step.

---

## The Missing Notes

Most notes have a sharp or flat between them.

Example:

```
A → A#/Bb → B → C → C#/Db
```

However, two pairs are naturally separated by only one half step:

```
B → C

E → F
```

These are called **natural half steps**.

There is no:

```
B#
E#
```

in the basic musical alphabet.

---

## On the Guitar

The guitar is a physical representation of the chromatic scale.

Each fret raises the pitch by one half step.

Example:

```
Low E String

Open
E

1st fret
F

2nd fret
F#

3rd fret
G
```

The fretboard is not a collection of numbers.

It is a map of musical relationships.
