## Objective

Begin seeing the guitar neck as a repeating musical pattern.

The goal is not to memorize isolated fret numbers.

The goal is to understand that the fretboard is organized by the same musical system we learned in Lesson 01 and Lesson 02.

---

## Lesson

A guitar fretboard is not random.

Every string follows the same chromatic sequence.

The open string determines where the pattern begins.

For the Low E string:

```
E F F# G G# A A# B C C# D D# E
```

For the A string:

```
A A# B C C# D D# E F F# G G# A
```

The pattern is identical.

Only the starting note changes.

---

## The Fretboard Repeats

The chromatic pattern continues across every string and every fret.

The guitar is built from repeating relationships.

A note is not important because of where it is located.

A note is important because of what it is.

The third fret of the Low E string is not simply:

```
3
```

It is:

```
G
```

The seventh fret of the Low E string is not simply:

```
7
```

It is:

```
B
```

The fret number tells us where our hand goes.

The note name tells us what we are playing.

---

## The Octave

When we reach the 12th fret, we return to the same note.

Example:

```
Open E

12th fret E
```

The note name is the same.

The pitch is higher.

This relationship is called an **octave**.

The 12th fret acts like a reset point on the guitar.

The same pattern begins again:

```
E F F# G G# A A# B C C# D D# E
```

The notes continue upward, but the relationships remain the same.

---

## Why This Matters

The fretboard can feel overwhelming when viewed as hundreds of individual locations.

A guitarist might see:

```
22 frets
6 strings
132 possible positions
```

A musician sees:

```
12 notes

repeating patterns

relationships
```

The goal is not to memorize the entire neck at once.

The goal is to recognize the patterns that already exist.
